#CSS

 - 含义:样式表
 - 引入方式
    - link标签 ------外联(推荐)
    - 在head标签中书写style标签------内嵌(做测试)
    - 在标签中每个都有style属性值:CSS代码------内嵌(一般不使用)
 - 书写规范
    - 选择器:你要为页面中的哪一块设置样式
        - 基础选择器
            - '*' 通配符选择器
            - 'tagname' 标签选择器，直接使用标签名
            - '.classname' class选择器,给标签添加一个class属性
            - 'idname' id选择器,给标签添加id属性
            - id名在同一个页面中只可以出现一次(规范)
        - 基础选择器优先级: id> class > tag >*
    - 样式书写规范
```css
    * {
       属性名: 属性值
    }
    div {
       属性名: 属性值
    }
    .table{
       属性名:属性值
    }
    #table{
       属性名:属性值
    }

 ```
